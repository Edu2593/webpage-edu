Computer-Aided Design

Model (raster, vector, 2D, 3D, render, animate, simulate, ...) a possible final project

Introduction

For the Computer-Aided Design assignment, I opted to use the Fusion 360 software, since this software is used by some friends of mine in their company and I have received good comments. Furthermore, for image editing, I decided to use GIMP and Inkscape, because they are open source. Also, I tried to use Corel Draw, but I got discouraged because I only had a 15-day trial.

Rasters & Vectors

Raster image files use pixels, and each of these contains color information, which makes them dependent on the resolution of the image. That is, a good quality image requires more storage space. On the other hand, vector image files only use points and lines, therefore, they are not dependent on the resolution. A simple application of this type of file would be a logo. However, a vector image can not store much color information, so they are not convenient for digital images. I found a video that better explains the difference in the following link.
I used GIMP and Inkscape to vectorize an image.

Step 1

Open GIMP

I opened an image in GIMP.

Step 2

Smooth

I smoothed the image with G'MIC, a plug-in for GIMP.

Step 3

Smooth

The filter to smoothen the image is Smooth [Mean-Curvature]. And the parameters are by default.

Step 4

Desaturate

The colors of the image are desaturated, to convert it to black and white. This is done to decrease the color information in each pixel.

Step 5

Save

The edited image is saved in .png format.

Step 6

Open Inkscape

The image saved in .png format is opened in Inkscape.

Step 7

Trace Bitmap

A bitmap is traced on the image.

Step 8

Trace Bitmap

The brightness of the image is cutoff, and the threshold is varied to obtain the desired black-white contrast.

Step 9

SVG

Finally the image generated by the trace of the bitmap is dragged. And you can change the size without altering the resolution. The image is saved in .svg format.

3D design

Fusion 360

I'm more familiar with Autodesk Inventor, but I want to try modeling with Fusion 360 from now on. The steps to start a model in Fusion 360 are simple. Below are the steps I followed to model a possible seat for my final project.

Step 1

Open Fusion 360

When opening Fusion 360, to maintain the projects in order, it is recommended to create a project folder.

Step 2

Name the project

Put the name you want to your project.

Step 3

Create a folder

You can create as many folders as you want. Is for maintaining order between the files. On the right side of the window, an untitled file is automatically generated. In it, you can start modeling your 3D design.

Step 4

New component

To start modeling an assembly, it is necessary to create a new component by right-clicking on the name of the file in the browser.

Step 5

Model

As my intention is to design the seat for my final project, I looked for ways to do it ergonomically and respecting the distances of the human body. So I downloaded a human model in 3D and inserted it to the design as a subassembly.

Step 6

Working plane

To make the sketch, you must first select the plane in which you will work. In my case, I had to create a plane from an angle in the CONSTRUCT option.

Step 7

Sketch

Once the plane is selected, the sketch is made. In the image, you can see all the commands that can be used to draw the sketch. When you finish drawing the sketch you must select the FINISH SKETCH option.

Step 8

Solid

When leaving the SKETCH, you can proceed to build the solid with the commands that are seen in the image.

Step 9

Solid

In Fusion 360 you can edit the assembly and parts in the same window. In the figure, you can see the first tentative design of the seat of my final project.

Step 10

Work options

Fusion 360 offers rendering, animation, simulation, and manufacturing tools in its platform. In my opinion, it is useful to have all these tools in a single environment.

Step 11

Render

En la imagen se puede ver un intento de renderizado del asiento.

Step 12

Animation

I still do not fully control the animation option, but I managed to do a zoom to the seat as an animation.

Remarks

Fusion 360

One of the things that encouraged me to use Fusion 360 for the FabAcademy is that it has the option of version control. As you can see in the image, you have access to all the versions. And each version shows details of the date, time and who edited it.
